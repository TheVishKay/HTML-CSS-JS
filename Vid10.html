Classes and IDs

ID= Unique identifier

Class= Predefined set of properties you want something to possess accordingly

You can use div(.)class1.c2.c3 and so on to directly give classes using emmet

You can also use div(or any other kind of block/ inline element)#ID to directly give ID using the emmet

This all is based on the assumption that the Classes are predefined. The ID will be the identifier set by you
