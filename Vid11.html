Entities :

Entities are basically stuff you need to enter to input special characters into your text which are mainly reserved by html like /p.

Have become mostly useless now but still can be used for margin handling and also for inserting special characters like nonbreakingspace (nbsp) with an ampersand sign before em.

  can be used to give non breaking space , mainly for margining for noobies

¥ for giving the yen character

£ for giving the pound character

©; for giving the copyright character

™ for giving the trademark character

​ for giving the empty character

Basically it is for special characters
